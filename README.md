# FUTURE_PE_01

## Task Name
AI Website Copy Generator for Local Businesses

## Internship
Future Interns – Prompt Engineering Track

## Business Selected
DreamRest Mattress Manufacturing

## Project Overview
This project demonstrates the use of Prompt Engineering to generate professional website copy for a mattress manufacturing company. The content was created using AI tools and structured prompts to produce website-ready copy that is persuasive, customer-focused, and suitable for commercial use.

## Objective
To create a complete website content package for a local business using AI-powered prompt engineering techniques.

## Prompt Used

Act as an expert website copywriter.

Create professional website copy for a mattress manufacturing company called DreamRest Mattress Manufacturing.

Generate:
1. Homepage Hero Section
2. About Us Section
3. Product Categories
4. Why Choose Us Section
5. Customer Benefits
6. Strong Call-to-Action Section

Tone:
Professional, trustworthy, customer-focused, and modern.

Target Audience:
Homeowners, hotels, hostels, hospitals, and furniture retailers.

The content should be persuasive, simple, and ready to use on a business website.

## Deliverables

- About Us Section
- Product Categories
- Why Choose Us Section
- Customer Benefits Section
- Call-to-Action Section
